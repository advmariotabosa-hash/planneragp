# PlannerAGP

## Descrição
Aplicativo de planejamento e gestão de projetos AGP.

## Tecnologias
- NestJS
- TypeScript
- Supabase

## Instalação
```bash
npm install
npm run start:dev
```

## Configuração
1. Clone o repositório
2. Instale as dependências
3. Configure as variáveis de ambiente

## Desenvolvimento
- Rode `npm run start:dev` para desenvolvimento
- Rode `npm run build` para produção
